# Gunner 3 remake

Try the game [here](https://androettop.github.io/gunner-3-remake/)

![screenshot](./screenshot.png)

## Controls

- Arrow keys to move
- Shift to jump
- Ctrl to shoot
- Keys 01234567890 QWERTYUIOPAS to change weapon
- Z to do a roll
